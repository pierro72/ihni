Lancement de l'application Paramètres envoyés

Lorsque un utilisateur lance une application depuis IHNI, ses informations sont envoyées en POST à l'adresse : /qub/{url_application}

Paramètre :

  • Methode : POST

Form Data :

  • "apiKey" : string (clé d'api de l'application lancée)
  • "user" : string (nom + + prénom)
  • "id_user" : int
  • "admin" : bool
  • "team" : string (nom de l'équipe choisie par l'utilisateur pour lancer l'application)
  • "id_team" : int (id de l'équipe choisie par l'utilisateur pour lancer l'application)
  • "role" : string (role de l'utilisateur dans l'équipe choisie pour lancer l'application)

Exemple :

apiKey:f3a7da7e66b0
user:Nils Langen
id_user:2
admin:true
team:CDS Java
id_team:6
role:utilisateur

Méthodes

api/team/{id} Retourne les infos d'une équipe

Paramètre :

  • Addresse :
    {id} int: id de l'équipe
  • Methode : GET
    apikey string: clé d'accès au service

Return : format JSON

  • {
    • "info" : obj Equipe {}
    • "users" : array [ Obj{
      • "user" : obj User {}
      • "role" : string
      }]
    }

Exemple :

/api/team/2?apikey=f3a7da7e66b0

api/user/{id} Retourne les infos d'un utilisateur

Paramètre :

  • Addresse :
    {id} int: id de l'utilisateur
  • Methode : GET
    apikey string: clé d'accès au service

Return : format JSON

  • {
    • "info" : obj User {}
    • "equipes_role" : array[ (liste d'objets contenant le lien équipe/rôle)
        {
      • "equipe" : obj Equipe {}
      • "role" : string
        • }
      ]
    }

Exemple :

/api/user/2?apikey=f3a7da7e66b0

api/alluser Retourne la liste de tous les utilisateurs

Paramètre :

  • Methode : GET
    apikey string: clé d'accès au service

Return : format JSON

  • [
      {
    • "user" : obj User {}
      • }
    ]

Exemple :

/api/alluser?apikey=f3a7da7e66b0
